## NOTES
================================================================================

### Architecture Decisions

- Hybrid deployment for HIPAA compliance
- All PHI processed on-premises only
- Cloud receives only sanitized/tokenized data
- Azure OpenAI for LLM (HIPAA BAA compliant)

### Key Constraints

- PostgreSQL 14+ required
- Python 3.11+ required
- Node.js 20+ required
- Next.js 16.1.6 (locked)
- React 19.2.3 (locked)

### Risk Mitigation

- MFA required for Manager role
- Audit logs are immutable (7 year retention)
- JWT access tokens expire in 15 minutes
- Refresh tokens rotate on use
