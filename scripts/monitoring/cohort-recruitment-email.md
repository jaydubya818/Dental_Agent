## Red Flags to Watch For

**Technical Issues:**
- Connection failures > 50%
- Incorrect revenue amounts
- Job failures
- Slow response times

**User Experience Issues:**
- Confusion about setup process
- Unclear instructions
- Missing error messages
- Unexpected behavior

**Business Issues:**
- Sellers concerned about privacy
- Revenue numbers don't match expectations
- Trust score doesn't increase as expected

**If any red flags appear:**
1. Document the issue immediately
2. Create a GitHub issue
3. Communicate with affected sellers
4. Fix ASAP before expanding cohort
